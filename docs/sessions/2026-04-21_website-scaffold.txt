─── DECISIONS + TRADEOFFS ──────────────────────────────────────────────────────

1. Framework: Next.js 14 (not 15) — because React 19 RC broke wallet adapter
   peer deps. Static export is supported equally in 14.

2. 3D hero: CSS/SVG animated orb (not React Three Fiber) — +0 bundle vs +~200KB
   for R3F. The visual is convincing enough without a shader.

3. Payment model: lifetime NFT pass (not subscription) — one-time tx, no
   recurring billing headache, better web3 narrative for hackathon judges.

4. Hosting: Walrus Sites + SuiNS (not Vercel) — strong dogfood story. Forces
   static export, which aligned with scope anyway.

5. SuiClient type: minimal SuiRpc interface (not @mysten/sui's SuiClient) —
   dapp-kit returns a subtly different type. A narrow structural interface
   with just the two methods we use avoids the drift.

6. Solana payment: register_pro_pass Anchor ix with CPI transfer
   — DECISION REVERSED. User feedback: this was scope creep. Reverting to
   plain SystemProgram.transfer. Pro-gating becomes client-side (localStorage
   + optional on-chain re-check from the extension).
